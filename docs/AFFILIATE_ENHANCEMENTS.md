# Affiliate System Enhancement Roadmap

> **Document Created:** January 6, 2026  
> **Status:** Planning  
> **Priority:** High

---

## Executive Summary

This document outlines proposed enhancements to the SVP Affiliate System to improve lead generation capabilities for affiliates, strengthen networking relationships, and increase value for both affiliates and Strategic Value Plus.

---

## Current System Overview

### Existing Features

| Area | Current Capabilities |
|------|---------------------|
| **Onboarding** | 9-step wizard (profile, business info, expertise, networking preferences) |
| **Networking** | Member directory, AI match recommendations, 1-to-1 meeting scheduling |
| **Referrals** | Track given/received referrals, affiliate and SVP referrals, pipeline status |
| **Deals** | SVP referral tracking with commission tiers (7%, 12%, 17%) |
| **Metrics** | Leaderboard, engagement score, badges, conversion rates |
| **Lead Tools** | Apollo Search (prospect finding), Supplier Search (ThomasNet) |

### Current Affiliate Journey

```
Register ‚Üí Onboarding Wizard ‚Üí Complete Profile ‚Üí Schedule 1-to-1 ‚Üí 
Submit Meeting Summary ‚Üí Exchange Referrals ‚Üí Track Deals
```

## Enhancement Categories

### 1. Lead Generation for Affiliates ‚≠ê HIGH PRIORITY

**Gap:** Affiliates can refer leads to SVP but have limited tools to generate leads for their own businesses.

#### Proposed Features

| Feature | Description | Effort |
|---------|-------------|--------|
| **Affiliate Lead Capture Page** | Personal landing page for each affiliate with their expertise, services, and a contact form | Medium |
| **Lead Sharing Pool** | Allow affiliates to share leads that don't fit their services with the network | Medium |
| **Warm Introduction Requests** | Let affiliates request introductions to specific companies/contacts from other affiliates | Medium |
| **Reverse Referral Matching** | AI suggests which affiliates might have leads that match your ideal client profile | High |

#### User Stories

- [ ] As an affiliate, I want a personalized landing page so prospects can learn about my services and contact me
- [ ] As an affiliate, I want to share leads I can't service so other affiliates can benefit
- [ ] As an affiliate, I want to request warm introductions to companies in another affiliate's network
- [ ] As an affiliate, I want AI to tell me which affiliates are likely to have leads that match my ideal client
